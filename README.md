# FileMisc FileManager - POC

* Read content from one file in the left listview and adapt the changes at the file chosen in the right listview.
* In this stage, the only need is to rename the file in the right listview to the same name as the file in the left listview.
* Other functions will be added. Some of them will be:
    * Manipulate SELinux security mechanisms.
    * Manage btrfs volumes.
    * Container management.
    * *E.g., everything where you want to inherit one or more parts of content from one field to the other is in the scope for this project. Only imagination sets the limit.*

For now, the simple POC of a file renamer will do.

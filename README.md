 # FileMisc poc
 * **Purpose:**
 * Read content from one file, in the left listview and adapt the changes at the file chosen in the right listview.
 *  In this stage, the only need is to rename file in right listview to the same name as the file in left listview.
 *  Other functions will be added. Some of them are 
    * Manipulate selinux security mechanisms. 
    * Manage btrfs volumes.
    * Container management. 
    * *Eg everythig where you want to inherit one or more parts of 
        content from one field to the other, is in the scope for this project. 
        Only imaginagination sets the limit.*
 
 * For now, the simple poc of a filerenamer will do.
